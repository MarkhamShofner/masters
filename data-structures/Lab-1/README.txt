Java Version - 8
IDE - NetBeans

Can run from the command line as well.
From the `TowersOfHanoi` directory
- `javac src/towersofhanoi/*.java`
- `java -cp ./src towersofhanoi.TowersOfHanoi string1 string2 string3`
  string1 is a flag (r or i) for solving the problem recursively or iteratively
  string2 is the number of discs you want to solve for
  string3 is the name of the output file to go in the `OutputMoves` directory
